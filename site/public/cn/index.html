学习如何使用

自动获取元数据

从 arXiv，doi.org，DBLP，IEEE，PaperwithCode，尽可能为每一篇计算机领域论文匹配精确的信息以及代码仓库。

评分、标记、笔记

多种方式对论文进行评价标记，包括打分，旗帜标记，记录自定义笔记，来记录一些方法俗称等，如 ResNet，以方便快速检索。

云同步 (BETA)

基于 MongoDB atlas，我们提供数据库云同步功能，跨 Win Mac 平台随处访问您的论文库。

使用标签和文件夹进行管理

为论文分配标签、文件夹，方便进行分类检索，比如为一次投稿组织一些引用。

导出引用

导出论文引用到 Bibtex 或者 Plain Text 格式，Word？不存在的。

跨平台

支持 macOS 和 Windows。
